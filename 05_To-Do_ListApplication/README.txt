To-Do List Application

Developed a feature-rich To-Do List application in C++.

Key Features:

Task Management: Add, delete, update tasks, and mark tasks as completed.
File Handling: Save and load tasks from a file to ensure data persistence.
User Interface: Command-line interface for user interaction.
Priority Levels: Assign and sort tasks by priority.
Due Dates: Set and sort tasks by due dates.
Search Functionality: Search for tasks by keywords.
Technologies Used:

C++ (classes, vectors, file handling, exception handling)
Algorithms for task sorting and searching
Project Highlights:

Designed and implemented using object-oriented principles.
Enhanced user experience with robust error handling and data validation.
